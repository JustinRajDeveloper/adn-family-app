📖 Setup Guide
🛠 One-time Supabase Setup (5 min)
1
Go to supabase.com → New Project → pick a name like "family-app"
2
In Storage → Create bucket called photos → set to Public
3
In SQL Editor → run this query to create the photos table:
CREATE TABLE photos (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  url text NOT NULL,
  uploader text DEFAULT 'Family',
  caption text,
  created_at timestamptz DEFAULT now()
);

-- Allow anyone to read & insert
ALTER TABLE photos ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public read" ON photos FOR SELECT USING (true);
CREATE POLICY "public insert" ON photos FOR INSERT WITH CHECK (true);
4
In Storage → Policies → add policy: allow public uploads to the photos bucket
5
Go to Project Settings → API → copy Project URL and anon public key
6
Paste both into the Connect Supabase form at the top — done! 🎉
